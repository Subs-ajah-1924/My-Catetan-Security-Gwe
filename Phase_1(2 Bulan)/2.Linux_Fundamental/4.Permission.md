**rwx(Read,Write,Execute)**
setiap file direktori di linux mempunyai  3 jenis izin:
- Read : hanya boleh dibaca
- Write : Hanya boleh di tulis
- Execute : Hanya bisa di jalankan 

**Owner | Group | Others**

- **Owner** adalah pemilik dari file/folder tersebut
- **Group** adalah user lain yang di beri izin oleh owner
- **Others** adalah user lain di sistem yang bukan owner dan bukan anggota group

misal cobak command ls -l,nko output e bakal koyok ngene:

(rw- | r-- | r-- ) iki podo wae koyok (Owner | Group | Others).maksudnya disitu adalah 
Owner hanya bisa read dan write file tersebut,sedangkan group hanya bisa read saja dan others hanya bisa read.


**chmod Command Line** digunakan unutk mengubah izin akses file atau folder.contoh:
chmod 644 test.txt -> ini akan mengubah izin dari file test.txt
nilai nilai dari izin rwx:
- r  = 4 
- w = 2
- x  = 1
Gabungan:
- 7 = 4  + 2 + 1 = rwx
- 6 = 4 + 2 = rw-
- 5 = 4 + 1 = r-x
- 4 =  r--
- 0 = ---
ada 1 command line chmod yang bahaya yaitu chmod 777.karena 7 artinya rwx,ini memungkinkan semua user dapat melakukan apapun di file/folder yang di pilih.

**chown Command Line** digunkana untuk mengganti pemilik file.contoh:
- chown owner_baru:group_baru test.txt

beberapa contoh command untuk mencari permission:
- find / -writable -type f 2>/dev/null  = untuk mencari file writable.iki artine error e bakal lungo ngo folder iku,folder iku koyok void gampangane
- find / -user root -perm -u=x 2>/dev/null = mencari file executable milik root
- find / -perm 777 2>/dev/null = cari file yang memiliki permission 777
