ada beberapa command line yang sering digunakan untuk security/hacking,misalnya:
cd -> untuk mengganti directori 
ls -> untuk memberikan output dari direktori yang digunakan / dipilih
ls -la -> untuk memberikan output semua file termasuk yang tersembunyi dari direktori yang digunakan / dipilih
pwd -> untuk mengetahui saat ini berada di direktori mana
find -> untuk mencari file file di seluruh sistem.contoh:
- -find /home -name nama_file(file)
- find /home -type d -name nama_folder(folder)
id -> kyok kartu nama nek  sistem
whoami -> ngei output kwe sopo nek sistem
cat -> kanggo moco isi semua file dengan cepat 
less -> kanggo moco isi file sitik wae
