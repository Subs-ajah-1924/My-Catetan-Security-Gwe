**Kenapa windows sering jadi target attacker?**
Windows adalah sistem operasi yang paling banyak di gunakan di seluruh dunia. Dibuat oleh microsoft yang berfungsi normal seperti sisrem operasi yang lainnya.**Windows** biasanya sudah terpasang default di setiap laptop dan memiliki banyak fitur fitur,seperti office,kemudahan gaming,dll.

Tetapi windows menjadi target serangan yang disukai oleh attacker(hacker) dikarenakan sistemnya yang mungkin bisa dibilang cukup mudah untuk di otak atik.karena windows sudah ada default di setiap laptop dan terlebih lagi penggunanya orang awam(**karena mereka beli laptop untuk menggantikanPC yang harus paham dulu untuk merakitnya**) bisa saja mengklik file file yang di kirim attacker.akeh wong do mileh ndonlot aplikasi bajakan ketimbang langganan aplikasi premium dan biasane nek aplkasine iku wes di tempeli virus dan malware heker e.

windows memegang prinsip **Backward Compatibility** yaitu windows versi baru harus bisa menjalankan software windows versi lama.contoh : aplikasi dari wondows xp harus bisa jalan di versi windows yang terbaru.hal ini berdampak pada code  dari sistem operasinya karena tidak menghapus fitur-fitur lama dan menggunakan service lama.karena masih menggunakan hal hal tersebut otomatis keamanannya mungkin kurang karena keamanan kode dulu itu jauh lebih beresiko di bobol. 

Hal lainnya adalah ekosistem malware yang sudah matang karena dikembangkan bertahun tahun.bahkan sekarang kita tidak perlu membuat malware sendiri karena biasa di jual di darkweb dan biasanya mereka menjual virus ransomware karena windows sendiri sering di gunakan untuk perkantoran,kerja dengan file file penting. malware/virus tersebut sudah sangat kompaltible dengan windows itu sendiri.**kenapa malware mudah dikembangkan untuk windows**?karena windows memiliki format executable yang jelas(.exe dan .dll) ditambah struktur yang konsisten rapi sehingga mudah di baca dan di manipulasi oleh attacker.

Windows biasanya digunakan di lingkungan enterprise(kantor,kampus,pemerintahan) hal itu mengungkan bagi attacker karena bisa memperoleh akses ke sistem internal sehingga bisa mengobrak abrik hahahahhaiwhha apsi
