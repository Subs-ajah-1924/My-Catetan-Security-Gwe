**User** adalah identitas yang digunakan seseorang / proses untuk login ke sistem,menjalankan aplikasi atau mengakses file,registry dll. setiap user pasti punya:
- Username
- Credential(Password/Key)
- SID(id unik yang biasanya di pakai pada pengecekan token)

initine user iku koyok identitas seng tak nggo karo iso di nggo service kanggo melakukan sesuatu nek sistem e.berikut beberapa status user:
- enable = bisa login
- disable = tidak bisa login 
- locked = salah password terlalu sering 
setelah berhasil login,windows akan menciptakan token untuk di berikan kepada user.token tersebut berisi SID user,SID group,privilege,intregity level.setelah token dibut,semua akses tidak menggunakan password lagi,tapi menggunakan token tersebut.
info menarik.misal menjalankan apps dengan "Run As Administrator" iku sebener e orang ngenggo user "Administrator" tapi nggowo token seng sijine yaitu token seng hak hak e mirip koyok user "Administrator".mudeng la ya.dadi sebener e token e iku ono 2 filtered dan Elevated token.seng Filtered iku koyok user biasa sedangkan seng Elevated iku token seng administrator e aktif

Jenis-jenis user di Windows berdasarkan Scope
- Local User yaitu user yang hanya berlaku di satu komputer saja dan disimpan di SAM (Security Account Manager).lokasi path e C : \Windows\ System32\ config\ SAM
- Domain user yaitu user yang berlaku di jaringan domain dan dikelola di Active Directory.biasanya user ini di gunakan oleh kantor atau perusahaan

**Built-in User Account**
- Administrator = memiliki akses penuh ke sistem
- Guest = memiliki akses yang sangat terbatas ke sistem 
- DefaultAccount = Digunakan oleh sistem internal
- WDAGUtilityAccount = keamanan Windows Defender Application Guard

**Group** adalah kumpulan user user.hak akses  akan diberikan ke Group lalu user user di dalam group tadi bisa menggunakan hak akses tersebut.hal ini meningkatakan efisien dan lebih aman.

**Built-in Group Utama**
- Administrators = Memegang full control sistem
- Guest = Akses sangat terbatas
- Users = User biasa
- Power User = Legacy(hampir tidak bisa di pakai) 
**Built-in Group Sistem**
- Backup Operators = berfungsi untuk backup & restore sistem
- Remote Dekstop User = berfungsi login via RDP
- Event Log Readers = Berfungsi untuk baca log
- Network Configuration  Operators = Berfungsi untuk mengatur jaringan
