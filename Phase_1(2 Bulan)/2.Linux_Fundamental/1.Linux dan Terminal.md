- **Linux**
**Linux** adalah sebuah kernel open source yang sangat fleksibel untuk di modifikasi.**Kernel** itu adalah semacam sistem untuk mengatur hardware supaya software bisa jalan(sistem yang berhubungan langsung dengan hardware).di dalam kernel biasanya ada pengatur cpu,ram,driver,dll.

ada juga namanya distro.**Distro** adalah OS yang menggunakan kernel linux.setiap distro pasti menggunakan kernel linux yang sama tapi dibedakan sedemikian rupa,seperti merubah tampilan,target user,fleksibilitas,keamanan,dll.beberapa contohnya : debian ,fedora ,arch ,silverblue,kali linux,dll.

- **Terminal**
**Terminal** adalah antarmuka Teks(CLI) yang memungkinkan user berinteraksi langsung dengan sistem operasi linux(kernel,tools,dll).cara berinteraksinya menggunkan syntax yang sudah tersedia tergantung jenis **shellnya**.syntax shell ini meningkatkan efisiensi ketimbang mengunakan GUI(aplikasi).

**shell** adalah adalah otak penerjemah syntax yang di input dari user.user menulis syntax,lalu diterjemahkan agar terminal paham,lalu shell akan mengeksekusi sesuai perintah.terdapat banyak jenis shell di linux,contoh bash,zsh,fish,sh,dll.setiap shell mempunyai syntax yang berbeda-beda tergantung target usernya,ada yang mudah,ada yang kompleks,ada yang simple.

Terminal hanya mengeluarkan output teks dari proses yang di eksekusi,sedangkan shell itu yang menerjemahkan inputan user dan mengirimnya ke kerne(sistem).kira kira begini alurnya:
- user melakukan input misal buka firefox
- shell akan membaca inputan teks tersebut 
- shell akan mencari program di sistem sesuai dengan inputan 
- program jalan 
- output dikembalikan lewat terminal 

terminal(CLI) itu lebih kuat dan efisien dikarenakan sifat terminal itu sendiri.di terminal input bisa dijadikan output,bisa di ulang ulang dan bisa di otomatisasi.

- Recon,scanning,enumerating.intine proses seng nyari celah dan mengecek keamanan
