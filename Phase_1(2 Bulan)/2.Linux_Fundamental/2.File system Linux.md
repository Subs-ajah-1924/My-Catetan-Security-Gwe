**PATH** adalah alamat lokasi file/folder di sistem linux.contoh home/user/Download/framwowo x bund tedi.mp4.itu menunjukkan lokasi file video tersebut di  folder home,lalu folder user,lalu dai folder Download.folder terahir adalah lokasi file tersebut.
terdapat beberapa jenis PATH,yaitu:
- path absolut,ini merupakan alamat lengkap dari file/folder yang pasti selalu di awali oleh "/" .contoh : /home/subahehe/Document/supersemar.txt
- path relatif:path yang tergantung pada posisi directori yang ditempati.contoh Document/supersemar.txt .berarti aku berada di direcotry /home/subahehe

**Directory /(Root directory)**
Root diroctory adalah folder paling tinggi di hirearki filesystem linux.terdapat juga namanya root user(superuser),root user adalah pemegang hak paling tinggi di sistem linux.dengan user root kamu bisa baca semua filr,ubah config,hapus/install apapun.   

**Isi dari directory / adalah**:
- **/bin(binary)**:tempat yang berisi command-command inti sistem dan harus ada bahkan saat sistem hanya bisa digunakan sedikit.contoh : ls,cp,mv,cat,bash
- **/sbin(system binary)**:hampir podo koyok /bin,tapi iki tools iku khusus kanggo sistem.contoh: mouth,reboot,ip,shutdown.intine seng berhubungan karo hardware.
- **/usr(Unix System Resources)**:data,program,libbrary tambahan.intine kwe iso nambah beberapa program,trs nambah command tah opo opo iku nek folder /usr.disini juga bersisi bebeapa data-data aplikasi milik user dll
- /home:digunakan untuk menyimpan data pribadi user dan tampat untuk mengatur config user.mirim C:\ tek nek windows
- /etc(Et Cetera):artinya dan lain lain,berisi semua konfigurasi sistem misal config user & group,nerwork,aplikasi,service,dll.
- /var(Variable):berfungsi untuk menyimpan data yang berubah ubah,seperti log sistem,cache,error,dll.
- /tmp(Temporary):digunakan untuk menyimpan file sementara yang biasa digunakan oleh program atau user untuk jangka pendek.biasanya akan hilang setelah laptop di matikan/di reboot.
- /root:home directoy khusus superuser.directori ini biasanya berisi config shell,command sbin(adminitrasi),backup config penting,dll.intine iki /home e superuser
- /dev(Device):merepresentasikan hardware sebagai file.isinya berbagai nama hardware yang di gunakan di kernel linux.
- /proc(Processes):berisi berbagai informasi sistem operasi secara realtime dalam bentuk virtual filesystem dan disimpan di RAM.mencakup detail tentang proses di kernel ,lihat device yang termouth,dll
- /sys(system):kombinasi /proc/sys/ isine file yang berisi informasi hardware,driver dalam bentuk teks agar bisa di lihat.file tersebut juga dapat di konfigurasi kernel,driver dll
- /lib(libraries):isi dari /bin dan /sbin
