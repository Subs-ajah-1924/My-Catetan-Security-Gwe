Proses dan Service di Windows dan Linux itu sama secara konsep,yaitu:
- Process = program yang sedang berjalan.setiap proses pasti punya user dan nanti di identifikasi oleh sistem
- Service = proses yang berjalan di latar belakang dan biasanya berjalan secara otomatis(Daemon kalau linux) 

yang menjadi pembeda proses di windows dan linux adalah caranya:
- Di windows identifikasi user itu menggunakan semacam token.token tersebut berisi SID,Privilege,dll.token tersebut dibuat setelah orang login ke username yang digunakan.sistem akan membuat 2 token yaitu  filtered dan elevated token.filtered token iku kanggo user biasa,isi tokene akeh batasan e.sedangkan seng elevated iku koyok root,dadi iso lapo lapo nek sistem e.batasan batasan iku nek windows jengene **integrity level**.integrity level adalah tingkat kepercayaan sistem terhadap user.in ada beberapa level yaitu:
	- Untrusted = digunakan untuk proses yang menggunakan login akun  anonim dan memblokir sebagian besar akses tulis
	- Low = utamanya digunakan sebagai pembatas internet dan membatasi akses tulis ke file file penting tertentu
	- Medium = ini adalah tingkat default untuk sebagian besar proses di sistem
	- High = Tingkat ini di berikan untuk proses yang dijalankan sebagai "Run As Administrator".dadi elevated token iku isi integrity level e iki
	- System = aku raja.tingkat ini berada di atas High yang berarti proses ini bisa mengotak atik sistem.ini biasanya  digunakan sebagai cadangan untuk sistem dan berbagai proses di dalam inti sistem
