File Permission adalah aturan yang menentukan siapa saja yang boleh melakukan apa aja terhadap file tersebut.contoh 
- Read = hanya boleh dibaca
- Modify = boleh edit dan hapus file
- Write = hanya boleh di tulis
- Execute = hanya bisa dijalankan
- Full Control = bisa melakukan semua  hal di atas

di windows permission tersebut disebut **NTFS**(New Technology File Sytem) seng fungsine kyok file system pada umumnya lah,ngatur data disimpan,ngatur akses,ngatur keamana.

coro ngeti permission e:
- GUI = bukak file manager -> klik kanan file -> properties -> security.nek kono bakal ketok kabeh,user e,trs permission e karo iso configurasi nek kono 
- CLI(Command Line) = menggunakan command icacls doc.pdf dan contoh outputnya seperti ini:
	.\doc.pdf Subahehe\sikroco:(F)
    NT AUTHORITY\SYSTEM:(I)(F)
    BUILTIN\Administrators:(I)(F)
    Subahehe\SubaheheBruhh:(I)(F)
    
    **F artinya full controll dan I artinya Inheritance,nama permissioan  tergantung inisial jenis permissionnya gampangnya**


Jika menggunakan GUI pasti terdapat beberapa istilah seperti:
- Inheritance(Turunan) =Permission file atau subfolder di dalam folder inherit akan mengikuti folder utamanya
- Allow x Deny = jika cheklist di allow,maka mengizinkan dan jika cheklsit ada di deny,maka tidak mengizinkan.misal permission group due ijin bukak tapi user e orak ndue ijen,tetep gak iso bukak selagi user iku gak due ijin.
